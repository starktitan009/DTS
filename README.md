
# 🚛 DTS App (V1) — Transport Management System

A simple **web-based Transport Management System** built using **HTML, CSS, and JavaScript**.
This application helps manage trucks, drivers, trips, alerts, and basic business data — all stored locally in the browser.

## ✨ Features

### 🚛 Trucks Management

* Add, edit, and delete trucks
* Track truck type, capacity, and status

### 👤 Drivers Management

* Add, edit, and delete drivers
* Store driver contact details

### 🛣️ Trips Management

* Create and manage trips
* Track:

  * From → To locations
  * Dates
  * Expenses (fuel, toll, food, etc.)
  * Revenue
* Basic trip listing

### 🚨 Alerts & Reminders

* Create alerts with:

  * Title
  * Description
  * Date
  * Priority
* View and manage reminders

### 📄 Documents (Basic)

* Store simple document entries (no file upload in V1)

### 💰 EMI Tracking (Basic)

* Record EMI-related entries for trucks

### 📝 Notes

* Add and manage notes

### 📊 Dashboard

* Basic overview of system data

---

## 🧠 How It Works

* All data is stored in **browser localStorage**
* No backend/server required
* Works completely offline

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)
* Browser localStorage

## 🚀 How to Run

1. Download or clone the project
2. Open `index.html` in your browser
3. Start using the app

---

## ⚠️ Limitations (V1)

* No backend (data stored only in browser)
* Data will be lost if browser storage is cleared
* No authentication/security
* Limited scalability
* No file upload support
* Edit functionality may have bugs in some sections
